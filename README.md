### Building and Running

On Linux :

sudo apt-get install libxrandr-dev libxi-dev

cd glsdk
../premake/premake4 gmake
make
cd ..
premake/premake4 gmake
make

On Windows :

cd unofficial-opengl-sdk
..\premake\premake4 vs2010
[open and build solution]
cd ..
premake\premake4 vs2010
[open and build solution]
